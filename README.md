TFM Desarrollo de Software con IA

Página web sobre cultura japonesa "Senda Nipona"

Filosofía de desarrollo: Agile, esto me va a permitir ir desarrollando y probando el proyecto en iteraciones, teniendo y progreso constante.

Comandos

```bash
# Instalar dependencias
npm install

# Configurar variables de entorno
cp .env.example .env

# Crear y poblar la base de datos
npx prisma migrate dev
npx prisma db seed

# Ejecutar aplicación en desarrollo
npm run dev
```
